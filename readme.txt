WooCommerce integration for Wuunder parcel delivery platform.

== Description ==

Wuunder Shipping integrates your WooCommerce store with the Wuunder parcel delivery platform. Wuunder is a shipping management system that connects multiple carriers.

**Key Features:**

* Easily add Wuunder's shipping methods to WooCommerce

**Supported Carriers:**

Through Wuunder's platform, you can access various shipping carriers including postal services and courier companies across different regions.

**Requirements:**

* WooCommerce plugin (required)
* Wuunder account and API key
* PHP 7.4 or higher
* WordPress 6.4 or higher

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/wuunder-shipping/` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to WooCommerce > Settings > Wuunder
4. Enter your Wuunder API key in the Settings tab
5. Configure available carriers in the Carriers tab
6. Set up shipping methods in WooCommerce > Settings > Shipping

== Frequently Asked Questions ==

= Do I need a Wuunder account? =

Yes, you need a Wuunder account and API key to use this plugin. Visit https://wearewuunder.com to sign up.

= Which carriers are supported? =

The plugin supports all carriers available through your Wuunder account. Available carriers vary by region and your Wuunder subscription.

= Does this work with WooCommerce blocks? =

Yes, the plugin supports both classic shortcode checkout and the new WooCommerce checkout blocks.
